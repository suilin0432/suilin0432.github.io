Simple Summary for (Chapter15)《Deep Residual Learning for Image Recognition》

(Chapter15)《Deep Residual Learning for Image Recognition》

笔记请见这里 笔记地址

PS: 仍然需要用户名和密码

PS: 关于三个问题:

  1. 文章总体上认为是因为 VGG 和 GoogleNet 中 plain module(非residual function) 对于 identity 的学习是不太好的简单的加深网络会导致网络的收敛变慢, 同时也提到了 梯度消失和爆炸问题的存在也有影响.
  2. (1). ResNet中: 虽然理论上直接使用几个非线性模块进行堆叠可以拟合复杂的函数, 但是文章中表明, 其对 identity 关系的学习并不是很好并且不容易优化, 加上类似 shortcut 分支成为所谓的 DAG 模块结构同样应该具有拟合复杂函数的能力, 并且能够更加容易优化. (2). 同时在GoogleNet则是认为加宽网络模块会带来性能的提升, 同时文章中认为是稀疏分布的叠加->形成密集结构, 有利于效率
  3. ResNet 以及 GoogleNet 都是使用 avg pool 代替了那两层 FC 进行的处理. 优点:
    • (1). 1*1 conv + fc 大幅减少了参数量
    • (2). global avg 可以接受任意规模的输入输出, 可以支持变尺度的训练
