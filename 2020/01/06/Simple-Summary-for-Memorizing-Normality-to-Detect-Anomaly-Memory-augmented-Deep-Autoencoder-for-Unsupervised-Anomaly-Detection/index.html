##《Memorizing Normality to Detect Anomaly: Memory-augmented Deep Autoencoder for Unsupervised Anomaly Detection》

笔记请见右侧地址: 笔记地址

问题:

  1. 个人认为 shrink 那个操作为什么有效的本质是因为 异常点 会和 normal 数据的 encode 出来的信息有比较大的差异的, 所以一般不会出现那种一个相似度非常高的情况出现吧, 这时候其他的带来的影响还是比较大的
  2. 个人认为更重要的点在于: 找相似度得到的信息我们期望的应该是重构出来的是一个能表达的事物, 但是这个重构出来的事物应该和异常点所表达的不同, 而不应该是异常信息从 memory 检索出来之后重构出来的信息无法表示 或者 异常信息所直接 encode-decoder 出来的信息是没有被很好学习到的(这一点和文章后面实验图所表示的意思是相似的, 但是认为检索过程中的数学形式应该还会有更精妙的方法达到这一目的)
