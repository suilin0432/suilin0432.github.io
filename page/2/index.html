Simple Summary for Representation Learning by Learning to Count

Posted on 2019-11-19

简单总结《Representation Learning by Learning to Count》

PS: 在论文总结中说的是本文学习在downsampled images与tiled images中数features数目并且使其在两者中满足等价关系.
PS: 另一个论文总结中说实质上本文受益的是: 在高层次上, 通常语义信息丰富、纹理信息复杂的norm都大

Introduction:

  1. 本文目的: 提出一个基于视觉原语的使用artificial supervision signal的方法(这里的artificial指的应该是人去定义的而不是人工标注的)去进行representation learning.
  2. supervised signal是通过equicariance relation获取的, 不需要人工的标注. 文章将transformations of image和transformations of representations相联系起来. 但是文章强调自己关注的找到满足这种关系的representation而不是针对特定representation的transformations. 本文在context of counting中使用两种transformations: scaling && tiling. 强调scaling利用的是visual primitives的珊瑚粮应该是与scale无关的, tiling则允许我们去同等看待整个图片中没个tile的visual primitives的总数. 提到了这两个transformations会被结合到一个约束中, 并用来使用一个contrastive loss去进行neural network的训练. 本文提出的方法在迁移学习中有很好的效果.
  3. 本文认为先前的自监督的工作表明了, 自监督学习的本质与最终要解决的问题最相关的自监督任务(pretext task)更可能构建出来好的representations. -> 本文的pretext task: counting visual primitives (这个自监督信号需要有辨别力的feature, 对分类是有帮助的, 同时其可以通过detection任务进行formulated).
  4. 为了获取对counting有利的supervision信号, 利用了如下的性质: 如果我们将image分割成了不重叠的regions的时候, 每个regions 中 visual primitives的数量加起来就是整个图片中visual primitives的数量
  5. 同时本文做出的假设是: 模型需要能够将image分离处high-level factors of variation,从而将orginal image与其regions之间的复杂关系转变为简单的算数关系(本文就是counting).
  6. 强调了虽然本文使用的是特定的两种变换的组合, 但是可以使用其他的变换的组合作为vision signals. 同样可以用于更大范围的任务之中.
  7. 本文贡献: (1). 引入了一个新的不使用手工标记数据的特征学习方法 (2). 提出利用counting作为自监督任务, 同时证明了其与counting visual primitives之间的联系(???) (3). SOTA效果

Method:

  1. Transforming Images to Transform Features:
    • (1). 提及到characterize a feature of interest的一种方式就是描述其是如何作为function随着input data的变化而变化的. eg: visual primitives的计数不应该被scale, 2D translation, 2D rotation变化所影响. 整体的magnitude of feautres应该大于region的.
    • (2). 本文, 作者会使用已知的input与output之间的关系作为superisory signal 去进行representation的学习.
    • (3). 首先定义了图片为 $x ∈ R^{mn3}$ 定义了一组image transformations $G={G_1, …, G_J}$ 每个变换 $G_j : R^{mn3}->R^{pq3}$ 将image作为输入, 并且将其映射到 p q大小的image上. 之后定义一个feature $φ:R^{pq3}->R^k$ 将transformed image映射到一个 k 维度的vector上. 最后, 定义一个feature transformation $g:R^k···*R^k->R^k$ 将J个features作为输入, 并将其映射到另外的一个feature中. 因此给定了image transformation family G 和 g, 通过后面的公式所定义的artificial supervisory signal学习feature φ: $g(φ(G_1◦x), …, φ(G_j◦x)) = 0$
    • (4). 在本作中, transformation family由downsampling operator D(下采样系数为2), tiling operation operator $T_j, j = 1, …, 4 $组成. 因此 $G={D, T_1, … ,T_4}$. 同样定义了 $g(d, t_1, …, t_4) = d - Σ_{j=1}^4t_j$ 这个公式可以被写成 $φ(D◦x)=Σ_{j=1}^4φ(T_j◦x)$
    • (5). 说这个关联也和equivariance(同变性, 指的是不要求CNN所强调的invariance不变性, 而是要考虑到坐标之类的信息的)有所关联. Equivariance通常被定义为一个给定feature的性质. 本文中通过固定transformations并寻找一个满足这些变换的representation来将上面的定义的逻辑反过来. 同时, equivariance被约束在应用在inputs和features的transforamtions上. (???有点迷)
  2. Learning to Count:
    • PS: 使用CNN去获取representations.
    • (1). 原则上可以去用一个large database中的图片进行训练, 使用基于上面counting公式的 l2 loss作为损失函数. 即: $l(x)=|φ(D◦x)-Σ_{j=1}^4φ(T_j◦x)|^2$ 但是认为loss总是会有 φ(z) = 0 的这个平凡解的. 为了避免这个情况的出现, 选择使用contrastive loss, 连接是知乎一个描述 本文中的 loss形式为: $l_{con}(x,y)=|φ(D◦x)-Σ_{j=1}^4φ(T_j◦x)|^2 + max{0, M-|φ(D◦y)-Σ_{j=1}^4φ(T_j◦x)|^2} , x\not=y$ 目的是希望能够强迫两个随机选择的不同的image的counting feature尽可能的不同. M取经验值为10.
    • (2). 结合结构图食用可以明白: 整个流程是: 1). 选取两张不同的图 2). 将一张图只进行downsample, 将另外一张图做downsample并且进行tiling 3). 使用共享参数的AlexNet进行2).中得到的图的处理 4). 每个后续接上几组全连接层进行处理, 得到对应的最后的vector 5). 进行Loss的计算, Loss分为两个部分 1. 进行了downsample以及tiling的图片中: 计算正常的counting loss的计算 2. 只进行downsample图片的最后结果和另外image的tiling计算结果进行max(0, M - counting loss)的计算(希望两个不同的image 的 intra-image 差异尽可能的大)
    • (3). Least effort bias: a bias of the system是: 可以通过学习尽可能少的对visual primitives进行count来达到基本loss(指的是$l(x)=|φ(D◦x)-Σ_{j=1}^4φ(T_j◦x)|$)的目标 因此很多的feature mapping会坍塌到0, 在imageNet、COCO等数据集上已经发现了1000个entry中只有几十个非零, 特征因此是稀疏的. 此外本文的transfer learning实验表明了hidden layers在一些benchmarks上表现很好. 但是本文的contrastive loss限制了least effort bias. 因为counting feature当可数的visual primitives很少的时候并不能很好的和其他图像进行区分. 因此contrastive项也可以看做是将不同图像(intra-)区分开来的作用.

Simple Summary for Deep Clustering for Unsupervised Learning of Visual Features

Posted on 2019-11-18

简单总结《Deep Clustering for Unsupervised Learning of Visual Features》

PS: Unsupervised 方法(Self-Supervised), 在综述贴中, 描述这篇文章描述的主要是 利用对feature representation的clustering去进行parameter的更新

PS: 确实本文的方法是无监督的方法, 通过聚类的方式产生了 伪标签去进行训练来得到好的representations, 但是最后的结果是怎么产生的??? 聚类类别明显是很多的啊, 最后是怎么对应到各个类别的呢? 而且怎么用来去做分割已经detection也不是很明白… 是后面要进行有监督的学习进行fine-tune吗? 那使用VGG什么的带来的效果看上去并没有那么好.

Introduction:

PS: 总体流程上个人认为还是整个pipeline, 在feature提取层上取出来对应的提取的特征, 用来进行无监督的分类, 然后给上一个伪标签, 后续进行到分类网络的最后进行使用这个伪标签的训练去反向传播(图并没有画最后面的一部分)

  1. 想要将clustering以一种end2end的方式融合到深度学习网络中, 并且是其能够基于大规模的数据集进行训练. 本文提出的是DeepCluster, 一个联合学习神经网络参数和对得到的features结果进行聚类标签的聚类算法. DeepCluster迭代式的使用标准聚类算法(K-means)对features进行聚类, 并使用subsequent assignments作为监督去进行network的权重的更新. 使用DeepCluster去无监督的对CNN在大数据集上进行训练. 最后可以达到SOTA效果.
  2. 从Fig 1中可以看出的本文的主要做法: 将没有标签的数据进行clustering做分类, 之后形成伪标签, 然后用伪标签去进行训练.
  3. 提到了传统的聚类算法通常是针对具有固定feature的线性模型进行的设计, 这些工作也很少能在学习features的同时很好的工作.
  4. 本文, 作者提出了新颖的聚类算法, 针对大规模的convnets的end2end的训练. 表明了模型能够使用聚类网络获取到有用的通用features. 本文的做法就是: 交替的进行(1). 对图像描述子进行分类 和(2).通过预测cluster assignments进行convnet的权重的更新. 文章说是为了简单才选择使用的Kmeans, 其他聚类算法也是可行的. 认为与self-supervised方法不同的是, clustering只需要很少的domain knowledge, 并且不需要inputs的特定signal.
  5. 本文同样通过修改实验规则证明了框架的robustness(修改了包括convnet网络结构以及数据集), 也强调了不仅仅能捕捉class-level information, 同时也能捕捉instance-level information.
  6. 主要贡献点: (1). 一个新颖的使用如k-means的基础聚类算法的end2end训练的unsupervised方法, 同时仅仅需要很少的额外步骤. (2). 在很多的unsupervised transfer任务上SOTA的性能. (3). 一个关于unsupervised feature learning的验证标准的讨论.

Method:

  1. 首先对一些流程、目标与定义进行了公式化的描述
  2. Unsupervised learning by clustering:
    • 首先强调了, 当对提取特征的network参数进行随机初始化的时候, 并不会产生好的features, 但是random features的性能要比瞎猜要好多了. 认为这个好的效果是由于conv本身的结构所带来的对input signal的强先验导致的. 本文的idea就是去利用weak signal去促进convnet的辨别能力.
    • 本文对convnet的output进行cluster, 然后使用后续的cluster分配的伪标签进行最后损失函数的更新, 这个过程会迭代的进行去进行features的学习然后进行分类
    • 这里指的convnet应该是没有接上后面进一步辨别网络的部分, 只是用来学习特征的, 文章使用k-means作为基本的聚类算法(但是作者观点是按照以往的工作, 这里使用什么聚类算法都是无所谓的)
    • k-means使用的分类标准是几何距离
    • 引入了一个 d * k (d是feature的维度, k是分类的类别)的centroid matrix C, 通过 $min||(f_{θ}(x_n))-Cy_n||^2_2$ 其中$y_n$的每一个entry都是0/1,且只能有一个1. 对这个问题进行优化来得到最佳的伪标签y_n以及矩阵C(虽然后续C没有使用)
    • 总体上就是, $min||(f_{θ}(x_n))-Cy_n||^2_2$ 去得到伪标签, 然后去进行分类任务的损失函数的更新进行网络参数的优化.
    • 但是提到了这么操作很可能会产生平凡解, 需要一定的操作去避免优秀解的退化
  3. Avoiding trivial solutions
    • 强调了平凡解(应该指的是所有的点都被分到了一个类别, 或者大部分的点在少数的几个cluster中的时候)的而存在, 不仅仅是对非监督训练neural network来说的, 对于任意联合学习discriminative classifier和labels的方法都存在平凡解的问题. 提到了discriminative clustering即使在线性模型中使用的时候也会出现这个问题. 提到了解决办法通常是: 约束或者惩罚每个cluster中的点的数量, 但是提到了因为涉及到了整个dataset中的计算处理, 不适合用于大规模数据集上的处理办法, 所以提出了一系列其他的想法:
    • (1). Empty clusters: 提到了出现所有点在一个类别的可能性之一在于没有限制empty cluster的产生. 提到了一个最常用的方法就是自动的去重新分配empty cluster -> 当一个cluster是空的时候随机选择一个非空cluster并且将这个非空cluster的中心添加一个小的随机扰动之后作为空集的新的中心, 然后重新的去将非空集合按照这两个中心点进行一次聚类
    • (2). Trivial parametrization: 提到了当分类时候将大部分的image分配到了少部分的cluster中时, 学习到的parameter θ会对这几个类别非常有辨别能力. 尤其是当极端情况下只有一个类别的时候, 这个时候分类类别将会百分百正确. 这种情况通常在数据类别不平衡的时候出现. 避免这个问题出现的一般策略就是从uniform distribution中进行image的sample, 或者使用伪标签.

  2. Generating an Image Pixel by Pixel:

    • (3). 因为图像是3通道的, 所以每个位置的值是由3个值所确定的, $p(x_i|x_{<i}) = p(x_{i,R}|x_{<i})p(x_{i,G}|x_{<i},x_{i,R})p(x_{i,B}|x_{i,R},x_{i,G})$ (也就是三个像素值之间也是有着条件概率关系的).
    • (4). 提到了在进行训练和验证的时候每个像素的值的计算都是并行的, 但是在进行图像生成的时候是sequential的去进行生成的
  3. Pixels as Discrete Variables:
    • 之前的方法对image中pixel值的预测是连续分布, 本文则把这个考虑成了离散分布(通过把上面(3)中提到的公式的每一个p项考虑成一个用softmax实现的multinomial)把值划分为256个离散的值就可以了, 认为这么做形式简单同时不再需要对其distribution的shape做特定的prior分布, 同时文章实验发现, 使用离散的分布式能产生一个更好的结果的.
  4. Pixel Recurrent Neural Networks:
    • (1). Row LSTM:
      • 强调了Row LSTM是单向的layer, 从上到下的一行一行去计算feautres. 计算是使用一维的1 * k 的 convolution进行的, 对一个像素x来说, 这个层次会对其上的一个倒三角型的区域进行计算(因为每个点会考虑几个像素, 然后上一层的点又会考虑再上一层的几个像素, 一层层就拓展到成了倒三角) k越大对每层捕捉的context越多. 认为共享的权重有利于保证translation invariance.
      • 整体流程是: 先对当前阶段的输入x_i用一个卷积进行处理, 对上一阶段的hidden state h_i-1进行一个卷积处理, 两者维度都是4h n n, 将两者相加, 然后用sigmoid激活函数(针对output、forget、input三个门), tanh函数(这怒地content gate这个门)处理一下的到了四个门的参数. 然后用四个门的参数、前一阶段的hidden state与前一阶段的cell state最后可以求出当前阶段的hidden state以及cell state.
      • 提到了, LSTM层中都会有 input-to-stage 以及 stage-to-stage 两个component, 这两个component共同决定了LSTM中的四个门. 为了提高计算的并行性, 对于input-to-state component首先会进行全图的input map的计算, conv是使用带有淹没的只考虑有效context的conv实现的, 最后得到的tensor的尺度是4h n n, 代表了input map上的每个位置的gate vectors. 之后再得到了前一阶段的 hidden-state 以及 cell-state 之后就可以计算当前阶段的两个值了.
      • 强调了针对对input-to-state component进行加权的conv(masked conv)是被提前计算好的(文章后面应该是有介绍的) -> LSTM的一次一次的迭代我们就可以得到了最后的state值了.
      • 提到了因为Row LSTM的receptive field的问题, 感受野是不可能去捕捉到整个全图的范围的
    • (2). DIagonal BiLSTM
      • 希望不像Row LSTM一样捕捉的是倒三角区域的感受野, 而是希望能够利用全图的context. 从两个角点开始, 每个角点都对其下方以及(右侧/左侧)提供context(中间点只会对下方提供context)
      • 提到了使用大的一维卷积都不会改变考虑的context的范围是全局的这一特点, 所以使用更大的kernel是没有很大帮助的. (…但是明明这个是local的…)
      • 具体操作就不看了
    • (3). Residual Connections: PixelRNN会训练到12层那么深 为了提高收敛速度以及解决反向传播的问题(爆炸、消失)所以使用残差连接. 就是每层简单的加一个shortcut
    • (4). Mask Convolution: 提到了每个pixel的h channel的features是会被分为3个部分的, 分别对应于RGB的通道. 当在预测R通道的时候, 只有上面和左面的pixel才能被作为context, 当预测G通道的时候当前的pixel的R通道也会额外被当做context, B通道同理. 为此提出了mask conv这个东西, 有两种mask的方式: 1). Mask A: 其只对PixelRNN中的第一个conv layer中使用, 也就对应着前面所说的, 只有相邻的pixel(这个应该根据文章不同的两个LSTM模块的设置去进行这个相邻的定义, Row针对的是上面的3个像素, DIagonal指的同样是按照其定义的相邻)的值以及当前pixel的自己通道之前的通道的值会产生作用.(实现方法是保证每次更新之后都将不应该放入连接的位置的值设置为0) 2). Mask B: Mask B会在除了input连接的第一个conv之外使用, 将连接放开为不限制一定要是预测完的channel才会进行连接.
  5. PixelCNN: 提到了Row LSTM和Diagonal BiLSTM layer度有一个潜在的无边界的依赖范围, 从而使得每个state的更新需要sequential的更新, 带来了额外的计算量. 认为一个直接额想法就是让RF变大, 但是不是没有边界的. 所以希望能够使用传统的CNN去进行边界的限定, 直接生成所有位置的features的计算. PixelCNN使用多个masked CNN layer去进行, mask仍然是为了防止看到future context. (从描述上来看个人是认为, 多个local的CNN的使用就不用每次都要从头考虑到全局的信息了, 但是生成的时候仍然是需要一个一个的生成的, 仍然是不能并行的)
  6. Multi-Scale PixelRNN: 多尺度PixelRNN是由一个unconditional PixelRNN和一个或者多个conditional PixelRNN组成的. unconditional PixelRNN首先用一个小一点的s s的image去进行生成, 然后conditional network将小的image作为输入去生成一个更大一点的n n的image

Simple Summary for Semi-Supervised Learning with Ladder Networks

Posted on 2019-11-12

简单总结《Semi-Supervised Learning with Ladder Networks》

Introduction:

  1. 本文将supervised learning与unsupervised learning在深度网络中进行结合. 提出的model被训练去同时减小supervised与unsupervised cost function. PS: 本文一个贡献就是对LadderNet进行改进, 从unsupervised变成semi-supervised的network.
  2. PS: 文章说的permutation invariant指的是排列不变性(个人认为因为是使用神经网络进行拟合的所以只是要求有这个功能, 并不要求有这个完全严格的输出值的限制), 应该是对pixel进行冲排列之后不会造成影响(所以因此不适合对conv进行这样的约束, 因为CNN考虑的是local信息)
  3. 提到了半监督学习中很多工作都进行了supervised和unsupervised的结合, 其中unsupervised任务包括了进行图像的重建或者能够将物体正确的分类.
  4. 提到了确实有同时使用无监督学习和监督学习的工作, 比如用无监督学习去辅助监督学习的工作(使用无监督学习进行预训练, 然后用监督学习进行fine-tune), 但是作者认为通常无监督和监督学习的任务是不同的, 使用无监督学习得到的特征可能并不适合于监督学习.
  5. 本文工作是基于Ladder network的, 提到了Ladder Network的辅助任务是在model的每层去denoise representation. 模型结构是添加了encoder到decoder的skip connection的autoencoder, 同时Ladder Network的learning task和denoising autoencoder类似, 但是其对每层都添加noise, 认为skip connection有助于减轻模型高层次represent details的压力. (PS: 这也就是为什么加noise了, 不然很容易就直接学习到identity关系了). 本文的改进点是 传统的Ladder network只使用了unsupervised学习, 本文进行了和supervised learning的结合
  6. 本文贡献:
    • (1). 能够与监督学习方法兼容: unsupervised部分关注与supervised learning找到线索的相关细节(注意强调了”相关”). 并且可以添加到现有的前馈神经网络中, 文章后续表明了, 可以先用supervised learning得到一个state-of-art结果之后进一步使用非监督学习进行性能进一步提升
    • (2). Scalability resulting from local learning(???没翻译明白): 在top层的supervised target之外还有per level 的 local unsupervised target, 认为会使层次更加适合深度网络.
    • (3). Computational efficiency: 正常而言需要几倍的训练时间去添加解码器, 但是本文通过更有效的方法只需要一个很小的时间倍数

Method:

  1. 本文首先从因变量模型角度对ladderNet进行解释.
    • (1). 提到了因变量模型是很有吸引力的半监督模型, 因为其可以结合监督学习和非监督学习. 其中唯一的区别就是label是否可以被观察到. 尤其是层次隐变量模型, 岂能通过底层来描述细节, 高层次去关注那些更加抽象的具有不变性质的feature以更好地处理任务.
    • (2). 讲述了一下DAEs和DSS两个模型的区别: DAEs: (给定观察量, DAEs从加噪版的观察量学习去重构观察量)x加噪声->x’->重构得到x^, 任务目标使 x^与x的MSE loss最小. DSS: (重构隐变量, 给定观察量) DSS训练一个映射函数f(x)->z, 来获得因变量z, 然后对z进行归一化之后使用一个降噪函数g(z_norm)->z^, 得到一个重构量z^, 任务目标是使 z^与z的 (这里强调了一下一定要对z进行归一化, 否则g会学习到恒等函数)
    • (3). LadderNet结构: 首先进行x的输入, 然后加噪进行一个映射关系的到第一个隐变量z^1, 然后对z^1加噪得到z^1’….直到z^n’, 然后会之后利用这个一层层的去还原(降噪)z^i, 同时每个z^i(skip connection)和z^i+1’都会共同来去学习还原z^i. 每层都是有损失函数的贡献的(z^i与z^i’的MSEloss) z^i是用另外的一个不加噪编码的分支得到的
    • (4). 本文的有监督部分是在加噪过程的最后的那个hidden layer后面利用这个representation去学习针对分类的任务, 对此进行有监督的训练, 无监督的训练指的是laddernet本身(也就是target为 加噪过程的降噪结果得到的还原隐变量和不加噪过程得到的隐变量的MSE loss)

PS: 个人认为本文的思路主要就是为了解决AutoEncoder自监督学习以及有监督分类的任务目的的不同, 自监督的特征会强调保留原有特征信息去用来重构, 但是分类任务通常需要抽象的语义信息, 所以本文对加噪encoder部分的最后进行监听, 希望能够使得encode特征是有用的, 同时又能够利用自学习(非监督学习)很好的约束优化(因为这时候还没有好的初始化方法去使得直接训练网络达到一个很好的效果)

PS: 具体的分析有点看不懂, 只是大致理解思路而已, 有需要的时候回来看后面

《Stacked Convolutional Auto-Encoders for Hierarchical Feature Extraction》

PS: Anto-Encoders 相关

Introduction:

  1. Abstract: 本文提出了一个 convolutional auto-encoder(CAE) 去进行无监督的feature learning, CAE的堆叠形成了一个CNN网络. 每个CAE都是用传统的不添加额外的regularization terms 的 on-line gradient descent进行训练. 认为Max-pooling layer对于学习 biologically plausible(生物上可解释性???) feature是至关重要的. 之后本文使用由CAE堆叠的CNN去得到了一个很好的性能(在MNIST和CIFAR10上)
  2. 认为unsupervised learning methods的主要目的是为了从unlabelled data中提取通用的有用的feature, 然后用来去进行检测并移除冗余仅仅保留最关键的鲁棒的有辨别力的特征部分.
  3. 提到了大多数方法都是基于encoder-decoder结构的 -> input会先转变为相对来说维度较低的特征空间(encoder), 然后将其expand去重新产生initial data(decoder). 一个layer被训练之后, 其code会被fed到下一个阶段去进行更高非线性依赖的构建.
  4. 提到了Auto-Encoder的思想, 这里略过. 提到了Denoising Auto-Encoder: 强调不添加任何的约束的时候, 传统的auto-encoders总会学习到indentity mapping. 一个规避方法就是使用probabilistic RBM方法, sparse coding 或者 denoising auto-encoders(DAs). DAs尝试着去重建noisy inputs. 会通过想原始clean input添加根据input image的特征得到的基于一个noise distribution中的随机变量 -> corrupted input.
  5. PS: Denoising AutoEncoder简单总结 具体总结见笔记

Method:

  1. Convolutional Auto-Encoder (CAE)
    • (1). 强调的一点就是 AEs 与 DAEs 都忽略了2D image的结构问题, 使用FC. 不仅仅对于图像的分辨率来说是不太可行的, 同时引入了参数的冗余, 使得每个feature都是全局的. 这与目前计算机视觉领域的趋势–利用局部特征不太符合.
    • (2). 本文使用的CAEs则就是之前提到的 -> 利用局部的特征. 整个处理的过程的数学公式都是类似的, 只不过是用卷积和激活函数代替一下. 同时为了减少参数自由度去使filter得到的是图片整体内一个相同类型的feature, 所以使用的bias是公共的, 而不是每个pixel一个. 激活函数选择的是 tanh. 使用 MSE 作为 Loss Function.
  2. Max-Pooling: 这里说max-pooling layer的作用通常是用来获取具有位移不变形的representation. 认为其有助于提高filter的选择性. 本文选择使用max-pooling的原因是希望feature detector能够变得更加的广泛被应用, 避免那些没有鲁棒性的琐碎的模式被学习(比如identity). 在reconstruction阶段, 系数的latent code 将会减少每个pixel进行decoding的filter的平均数量, 使得filter更加general.
  3. Stacked Convolutional Auto-Encoders (CAES): 将一些AEs进行堆叠得到一个深层次的结构就是CAES了.

简单总结《Learning Deconvolution Network for Semantic Segmentation》

Introduction:

  1. 本文主要思想就是学习一个深的deconv network.
  2. 强调了传统方法会出现的一些问题: 对待极端大物体会出现属于同一个物体pixels的不一致lable(就是物体太大了, 不能完全辨认, 被network辨别成了多个不同的物体), 对待小的物体可能会被忽略掉. 以及物体的细节结构很容易被丢失或者平滑掉, 因为输入到deconv层的label map相对来说太粗粒度了, deconv过程也相对太简单. 强调了FCN在得到1616的label map之后直接就deconv到原图尺寸, 而没有用大规模数据集去训练一个deep deconv network, 也因此使得重建一个关于物体边界的highly non-linear structure 变得十分困难, 因此当前很多工作使用CRF去解决这个问题. **PS:总结起来就是 目标的细节结构容易缺失, 边缘信息保留不够好 认为原因就是 -> FCN得到的label map过于粗糙, deconv过程太过简单(所以本文基于这个观点进行了问题的解决)***
  3. 本文使用了不同的策略, 贡献如下:
    (1). 学习deep deconv network, 包括了deconvoluiton, unpooling以及rectified linear unit layers.
    (2). 训练的网络被应用在individual object proposals上去获取instance-wise segmentations, 与最终的final semantic segmentation相结合.
    (3). 达到了一个很好的效果

Method:

  1. architecture: 整个网络就是由 (1). convolution part (2). deconvolution part 两部分构成的. 强调conv part就是将原图像进行特征提取, deconv part就是将提取到的信息用来进行object segmentation的生成. 最终的输出是和原图像相同尺度的probability map.
  2. conv part使用 VGG 16, deconv part是 conv part的镜像结构, 但是操作就取而代之的使用deconv, unpooling, rectification layers等.
  3. unpooling:
    • (1). 关于pooling, 强调了虽然是为了过滤掉没有必要的多余的”噪音”从而对classification任务引入了鲁棒性所以达到了很好的效果, 但是pooling过程中, 空间语义是丢失了的, 可能会对精确地定位产生了大的影响. 所以本文提出了在deconv part 中的 unpooling layers, 其记录了当时pool的过程中选择了留下了各个感受野中的哪个元素的坐标, 然后在unpooling过程中在那个记录的位置进行信息的填充.
  4. deconv:
    • PS: 本文所指的deconv部分, 并不是传统意义上的反卷积结构, 而是将unpooling得到的系数的activations值通过类似convlution的方法去进行处理.
